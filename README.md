# SkyTalk

SkyTalk — современный мессенджер для Android и веба.

## Возможности

- 💬 Личные сообщения
- 👥 Групповые чаты
- 📷 Отправка фотографий и файлов
- 🔔 Уведомления
- 🌐 Веб-версия
- 📱 Android-приложение
- 🔒 Безопасное общение

## Статус проекта

🚧 Проект находится в активной разработке.

## Цель проекта

Создать удобный, быстрый и современный мессенджер для общения.
